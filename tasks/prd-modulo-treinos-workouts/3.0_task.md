Criar os componentes reutilizáveis para o formulário de criação/edição de treino: ExerciseSelector (modal para buscar e adicionar exercícios do catálogo) e WorkoutExerciseRow (linha editável de exercício no formulário).

</skills>

<requirements>

- WorkoutExerciseRow: nome do exercício, sets alvo, reps alvo, descanso, notas, botão remover
- Drag-and-drop para reordenar exercícios
- Validação: nome obrigatório, sets mínimo 1
