## Visão Geral

Criar a página de listagem de treinos do usuário com busca por nome, filtro por tipo e cards visuais para cada treino.

<skills>
### Conformidade com Skills Padrões

- **ui-ux-pro-max**: Cards com hover states, loading skeletons, empty states
</skills>

<requirements>
- Lista de treinos do usuário logado
- Busca por nome (debounce 300ms)
- Filtro por tipo (strength, cardio, hiit, flexibility, mixed)
- Cards exibem: nome, tipo, número de exercícios, duração estimada, data
- Loading skeleton durante fetch
- Empty state quando não há treinos
- Link para detalhes do treino
- Botão "Novo Treino" → /workouts/create
- Responsivo: 1 coluna mobile, 2 tablet, 3 desktop
</requirements>
