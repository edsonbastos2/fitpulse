Atualizar o componente `ExerciseGrid.vue` para usar grid de 2 colunas no mobile/tablet e 3 colunas no desktop, integrando o novo `ExerciseCardCompact`.

<skills>
### Conformidade com Skills Padrões

- **ui-ux-pro-max**: Grid responsivo com breakpoints consistentes, gap adequado para touch targets
</skills>

<requirements>
- Grid de 2 colunas para telas < 1024px (mobile + tablet)
- Grid de 3 colunas para telas ≥ 1024px (desktop)
- Gap de 8px no mobile, 16px no desktop
- Usar `ExerciseCardCompact` ao invés de `ExerciseCard`
- Manter animações de fade-in com delay escalonado
- Manter `role="list"` e `role="listitem"` para acessibilidade
- Suportar estado de loading (skeleton grid)
</requirements>

## Critérios de Sucesso

- Grid exibe 2 colunas em viewport 375px e 768px
- Grid exibe 3 colunas em viewport 1024px+
- Cards renderizam sem overflow ou corte
- Gap consistente entre cards (8px mobile, 16px desktop)
- Animações de fade-in funcionam corretamente
- Acessibilidade preservada (screen reader lê lista corretamente)
