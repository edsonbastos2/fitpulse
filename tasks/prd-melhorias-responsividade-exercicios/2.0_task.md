Criar o componente `ExerciseFilterChips.vue` — uma row horizontal de chips de filtros ativos com scroll horizontal, substituindo a sidebar lateral no mobile/tablet (<1024px).

## Detalhes de Implementação

Consultar `techspec.md` → seção "Componentes Novos" → `ExerciseFilterChips.vue`.

Interface de chips:
```typescript
interface FilterChip {
  key: keyof ExerciseFilters  // 'difficulty' | 'muscleGroup' | 'equipment' | 'type'
  label: string               // 'Difícil', 'Peito', 'Barra', etc.
  value: string               // valor do filtro para remoção
}
```

Estrutura base:
```html
<div class="flex items-center gap-2 overflow-x-auto pb-2 hide-scrollbar">
  <button @click="openDrawer">
    <FunnelIcon /> Filtros
  </button>
  <button v-for="chip in activeChips" :key="chip.key" @click="removeChip(chip.key)">
    {{ chip.label }}
    <XMarkIcon />
  </button>
</div>
```

Reutilizar `ExerciseFilterContent` dentro do drawer (já implementado).

## Critérios de Sucesso

- Chips exibem filtros ativos corretamente (dificuldade, músculo, equipamento, tipo)
- Clicar no X remove o filtro e atualiza grid
- Botão "Filtros" abre drawer com opções completas
- Drawer fecha ao clicar fora, pressionar Escape ou clicar em X
- Scroll horizontal funciona quando há muitos chips
- Componente oculto no desktop (≥1024px)
- Foco trap funciona no drawer (Tab não escapa)
- Touch targets de 44x44px em todos os chips
