# Tarefa 1.0: Criar componente ExerciseCardCompact (card horizontal)

<critical>Ler os arquivos de prd.md e techspec.md desta pasta, se você não ler esses arquivos sua tarefa será invalidada</critical>

## Visão Geral

Criar o componente `ExerciseCardCompact.vue` — um card horizontal compacto com thumbnail/ícone à esquerda e conteúdo textual à direita, otimizado para grid de 2 colunas no mobile.

<skills>
### Conformidade com Skills Padrões

- **ui-ux-pro-max**: Touch targets mínimos de 44x44px, contraste WCAG AA, tipografia legível (mínimo 14px no mobile), line-clamp para truncamento
- **web-design-guidelines**: Revisar acessibilidade do card (aria-label, role, focus-visible)
</skills>

<requirements>
- Card horizontal com altura fixa (~96px mobile, ~112px desktop)
- Thumbnail/ícone de músculo à esquerda (64x64px)
- Nome do exercício com truncamento em 2 linhas (line-clamp-2)
- Badges compactos: dificuldade + 1 grupo muscular visível
- Link clicável em todo o card (NuxtLink)
- Estado de hover com elevação (desktop apenas)
- Suporte a ícones de dificuldade (Fire/Bolt/Sparkles)
- Herdar design system existente (card-hover, text-gradient, etc.)
</requirements>

## Detalhes de Implementação

Consultar `techspec.md` → seção "Componentes Novos" → `ExerciseCardCompact.vue`.

Estrutura base:
```html
<NuxtLink :to="`/exercises/${exercise.id}`" class="block">
  <div class="flex items-center gap-3 h-24 p-3 card card-hover">
    <!-- Thumbnail/Icon (w-16 h-16) -->
    <!-- Content (flex-1 min-w-0) -->
  </div>
</NuxtLink>
```

Usar classes Tailwind: `flex`, `items-center`, `gap-3`, `h-24`, `min-w-0`, `line-clamp-2`, `truncate`.

## Critérios de Sucesso

- Card renderiza corretamente em grid de 2 colunas (375px viewport)
- Nome do exercício truncado em 2 linhas sem overflow
- Ícone de dificuldade exibe cor correta (verde/fácil, amarelo/médio, vermelho/difícil)
- Card clicável navega para `/exercises/[id]`
- Hover funciona no desktop (elevação/sombra)
- Touch target mínimo de 44x44px no mobile
- Score Lighthouse acessibilidade 90+
