# Tarefa 4.0: Extensão do `useAuth`

<critical>Ler os arquivos de prd.md e techspec.md nesta pasta. Se você não ler esses arquivos, sua tarefa será invalidada.</critical>

## Visão Geral

Estender o composable `useAuth` existente para buscar e integrar roles do usuário junto com seu perfil. Isso conecta a camada de banco de dados (tabela `user_roles`) com o composable `useRoles`, garantindo que os roles sejam carregados automaticamente quando o perfil do usuário é fetchado.

## Detalhes de Implementação

Consultar a **Tech Spec** nas seções:
- **Interfaces Principais** → código de exemplo de `fetchRoles()` em `composables/useAuth.ts`
- **Fluxo de Dados** → passo 2: "Fetch de Roles"

A função `fetchRoles` deve fazer:
```ts
const { data } = await supabase
  .from('user_roles')
  .select('roles(slug)')
  .eq('user_id', user.value.id)
```

E popular o mesmo `useState('user-roles')` que o `useRoles` usa, garantindo estado compartilhado.
