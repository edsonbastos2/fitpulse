# Tarefa 7.0: Página `access-denied`

<critical>Ler os arquivos de prd.md e techspec.md nesta pasta. Se você não ler esses arquivos, sua tarefa será invalidada.</critical>

## Visão Geral

Criar a página `pages/access-denied.vue` que informa ao usuário que ele não tem permissão para acessar determinado recurso, explica o motivo de forma clara e oferece caminhos de ação (upgrade de role, voltar para dashboard). A página deve seguir o design system existente do FitPulse (tema escuro, gradientes, glassmorphism) e ser acessível.
