Criar a migração SQL que estabelece toda a estrutura de banco de dados para o sistema de roles: tabelas `roles`, `user_roles`, `role_permissions`, `trainer_students`, dados iniciais (seed), RLS policies para isolamento de dados, e migração automática de usuários existentes para o role `user`.
