# Tarefa 5.0: Middleware `role-guard`

<critical>Ler os arquivos de prd.md e techspec.md nesta pasta. Se você não ler esses arquivos, sua tarefa será invalidada.</critical>

## Visão Geral

Criar o middleware Nuxt `role-guard.ts` que valida se o usuário possui os roles necessários para acessar cada rota. Este middleware funciona em cadeia com o `auth.ts`: primeiro o `auth` verifica se o usuário está logado, depois o `role-guard` verifica se ele tem permissão de role para aquela rota.
